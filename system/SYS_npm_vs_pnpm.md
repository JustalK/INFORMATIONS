# Npm vs Pnpm

Benchmark : http://www.tiernok.com/posts/2019/faster-npm-installs-during-ci/

## pnpm is faster than npm

You can install it as follow : https://github.com/pnpm/pnpm

```
npm install -g pnpm
```

## Replacement commands

```
npm install
pnpm install
```

```
npm i <pkg>
pnpm add <pkg>
```

```
npm run <cmd>
pnpm <cmd>
```
